
# Mommy Client

Custom discord bot for https://discord.gg/mommy (65k+ members)
## Authors

- [@AutoKill](https://www.github.com/autokill)
## Features

- Ticket System
- Giveaway System
- Leaderboards (daily, weekly, monthly, all time)
- Private VC System
- AFK System
- Staff Applications
- Suggestions
- Confessions
- Skullboard
- Colors Panel
- Tips
